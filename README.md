# 🔥 Sobre o Projeto

A **Seleção & Brasa** é uma boutique premium de carnes localizada em Salto (SP), e este projeto foi desenvolvido para criar uma experiência digital sofisticada, moderna e focada em conversão.

Mais do que um simples catálogo online, o objetivo foi transformar a experiência de compra em algo visualmente premium, intuitivo e humanizado.

O sistema permite que o cliente:
- visualize os produtos
- adicione itens ao carrinho
- acompanhe os valores
- finalize o pedido diretamente pelo WhatsApp da loja

Tudo isso mantendo:
- identidade premium
- experiência gastronômica
- navegação elegante
- responsividade completa

# ✨ Funcionalidades

## 🛒 Carrinho de Compras
- Adição dinâmica de produtos
- Controle de quantidade
- Atualização automática de valores
- Resumo do pedido

---

## 📱 Finalização via WhatsApp
Ao finalizar o pedido:
- o sistema gera automaticamente uma mensagem personalizada
- inclui produtos selecionados
- quantidades
- valores
- total da compra

O cliente é então redirecionado diretamente para o WhatsApp da loja.
