## Sobre o Projeto

Cardápio digital responsivo e sofisticado desenvolvido para a **Seleção & Brasa**, uma boutique de carnes premium localizada em Salto (SP).

O projeto transforma o conceito tradicional de açougue em uma experiência gastronômica digital, apresentando os produtos com imagens de alta qualidade e descrições atrativas que induzem o cliente à compra,  sem exibir preços.

## ✨ Funcionalidades

- 📱 **Design responsivo** - 2 colunas no celular / 4 colunas no desktop
- 🖼️ **Grid de imagens** com atributos `alt` descritivos para acessibilidade
- 🍽️ **Categorias organizadas** - Carnes, Massas, Bebidas, Pimentas, Geleias, Risottos, Vinhos
- 🎨 **Identidade visual** com as cores da marca: #4A2A1B, #A95427, #F7EFE6
- ♿ **Acessível** - Compatível com leitores de tela (SEO otimizado)

## 🛠️ Tecnologias Utilizadas

- HTML5 semântico
- CSS3 (Grid e Flexbox)
- Design Mobile-First
- Google Fonts (**Playfair Display** + **Lato**)
- Acessibilidade (ARIA labels, alt descritivos)
